Indian Railways App



Login Screen

The app starts with the login screen. You can login as admin, or as the worker to input data.



Search Page

When you login as a worker, you see the search page where you can search the records for a particular employee.


Editing

You can edit a particular record field and then it will be sent for approval.


Printing Summary

You can print all the feedback for a particular employee or you can print an individual feedback case.

Printing Individual Case

Admin View

In the admin view you can see the changes made as approval cards and can approve them.
